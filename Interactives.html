Interactive Explorations

Mapbox Base Map Design

Using Mapbox Studio and Carto CSS, I was tasked with creating a slippy base map based on Kevin Lynch's Image of the City. I created symbol specifications from Lynch's original maps and then translated them to Carto CSS. Symbolization is customized by zoom level.

Leaflet Syrian Refugees

This interactive map utilizes the Leaflet JavaScript library to display time series data. The time series depicts Syrian refugee admittance by country by month in 2015 and overlays a choropleth map of showing Syrian refugees as a percent of the host countries' population.

Explore this map here! Best if viewed in Firefox.

D3 Global Immunizations

This interactive map utilizes the D3 JavaScript library to create a coordinated visualization of global immunizations. The map and graph below display global vaccination rates by country. Countries highlight across the map and graph as you hover!

Explore this map here! Best if viewed in Firefox.
