Mapbox Base Map Design

Using Mapbox Studio and Carto CSS, I was tasked with creating a slippy base map based on Kevin Lynch's Image of the City. I created symbol specifications from Lynch's original maps and then translated them to Carto CSS. Symbolization is customized by zoom level.
